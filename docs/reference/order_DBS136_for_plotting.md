# Return order for DBS136 plotting

Returns the 160-element character vector mapping the 136-row DBS catalog
into a 160-position layout (10 groups of 16 for 10 heatmap images). 24
entries are not in the DBS136 catalog and serve as NA placeholders.

## Usage

``` r
order_DBS136_for_plotting()
```

## Value

A 160-element character vector of row names for DBS136 heatmap layout.
